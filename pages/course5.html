Full Stack Development - Backend Python Django

Course Objective

The objective of this course is to equip students with the necessary skills to design, develop, and deploy Full Stack Web Applications using modern technologies like Django, Bootstrap, and MySQL. Students will learn to implement best practices in backend development, database management, and frontend design, while gaining experience in version control through Git. The course also aims to teach the fundamentals of Test Driven Development (TDD) to ensure high-quality code and robust applications. Finally, students will be introduced to the principles of continuous integration and deployment, enabling them to confidently deploy applications to cloud platforms.

Course Modules

Module 1: Frontend Fundamentals

HTML, CSS, and JavaScript Basics, Responsive Web Design, and Frontend Frameworks (React or Angular)

Module 2: Backend Fundamentals with Python

Python Programming Basics, Object-Oriented Programming, Database Design and SQL

Module 3: Django Framework

Django Models, View, Templates, Forms and REST Framework (DRF)

Module 4: Full Stack Project

Project Planning and Design, Backend Development, Frontend Development and Deployment

Course Outcome

Students will be able to develop and deploy fully functional full stack web applications using Django for backend development and Bootstrap for frontend design. They will have proficiency in managing databases with MySQL and effectively using Git for version control and collaboration. Students will also be able to implement Test Driven Development (TDD) practices to ensure the reliability and quality of their code. Additionally, they will gain hands-on experience in deploying applications to cloud platforms and setting up CI/CD pipelines for continuous integration and delivery.
